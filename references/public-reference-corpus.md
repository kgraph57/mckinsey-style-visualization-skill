# Public Reference Corpus

This corpus lists public strategy and executive-insight sources to study for structure, argument flow, and visualization patterns. Do not copy proprietary text, layouts, images, or branded assets. Use these sources to identify reusable patterns, then create original slide specs with explicit assumptions.

## Extraction Rules

When studying a source, extract only reusable structure:

- the strategic question the visual answers
- the visual pattern type
- the relationship between headline, chart, annotation, and implication
- how assumptions or uncertainty are disclosed
- the data density and hierarchy
- the executive decision the visual supports

Never extract:

- exact slide text beyond short titles needed for citation
- visual styling that implies affiliation
- proprietary data not intended for reuse
- logos, branded templates, or copyrighted images

## Reference Notes

Use this corpus to improve the skill's standards, not to imitate a firm. Public references are learning material for pattern recognition. Final outputs must be original, source-aware, and non-affiliated.
